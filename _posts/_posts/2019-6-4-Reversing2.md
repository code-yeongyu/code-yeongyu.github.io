# What is hex code?
The file that we can execute is a set of assembly instructions, which means it's a binary file.  
It would be really hard to understand the meaning if we have to only use the binary.  
So, hackers often convert the binary into hex, cuz it's much easy to read due to the length reduces.  

# What is Patch ?
Modifying the file of some programs or the memory of the process called the patch.  
The general example of patch would be the update system of operating system.
